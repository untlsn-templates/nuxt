# My Nuxt 3 Starter

Look at the [Nuxt 3 documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

# Additional to default nuxt starter
- [UnoCss engine](https://github.com/unocss/unocss) - The instant on-demand atomic CSS engine. Treat like Tailwind

## Setup

Make sure to install the dependencies

Pnpm if preferred but other (like npm and yarn) can be used as well

```bash
pnpm install
```

## Development Server

Start the development server on http://localhost:3333
Port on convig was change from 3000 to 3333

```bash
pnpm dev
# if port 3333 is taken, you can use other port like 3030
pnpm dev --port=3030
```

## Production

Build the application for production:

```bash
pnpm build
```

Locally preview production build:

```bash
pnpm preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
